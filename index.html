How do stable relationships emerge from chaos?

Relational networks are an analytical tool for exploring how networks of relationships change over time. Think of them as graph theory with a play button.

These networks are defined as follows:

  • Let there be a set of N nodes.
  • Let each node be capable of transmitting packets of information to other nodes.
  • Let each node choose which other node or nodes it will transmit to according to some probability distribution.
  • When a node sends or receives a packet, let its probability distribution be transformed in some way.

It is my hope that by studying this class of networks we might gain a deeper understanding of systems in which discrete entities dynamically share information.
A simple relational network. Each node has a "stability" value, indicated by its size. A node with high stability will always send its packet to the same place in the grid, while a node with low stability will randomly rotate where it sends it packet. Lines are drawn to illustrate the formation of stable edges within the network.
